Changing to NexT Theme

Posted on 2019-03-10 Edited on 2019-10-11 In Computer Science

Hexo is a extensive blog platform. There are tons of elegant themes for it and you can even create your own one! I use NexT theme for this site. In fact the Hexo official site keeps a collection of themes. In this post I am showing your how to change the default theme to NexT

Get the Theme Package

  • Installing the Latest Version

    Since Git is installed if you followed the tutorial in this blog, cloning the latest version the most convenient way. Change working directory to your local blog folder and get the newest one

  • Installing the Stable or a Custom Version

    However, you are always free to decide which version to install. This is simple as well.

    1. Visit the NexT Release Page.
    2. Download the version you favor.
    3. Make an empty folder named next in theme subdirectory.
    4. Unzip the downloaded assets to the next folder you have just created.

Enable the NexT Theme

  1. Configure Site _comfig.yml File

    Like all Hexo themes, after you download it, open site configuration file, find theme section, and change its value to next

  2. Choosing between Schemes

    The NexT theme contains four different styles that you can pick: Muse, Mist, Pisces and Gemini

    You can open the _config.yml file in the NexT theme folder, locate scheme key and comment/uncomment them.

  3. Check the Theme and Scheme

    After choosing a specific scheme, let us check it out. You may need to clean the Hexo cache to avoid any potential conflicts.

    Your site is available at http://localhost:4000.
